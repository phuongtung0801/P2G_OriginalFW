◆ radar_set_duty_cycle()

uint16_t radar_set_duty_cycle ( Radar_Handle_t  device,
const uint8_t  flag 
)

This function enables/disables the duty cycle by turning PLL/BGT ON/OFF.

The duty cycle is enabled in order to save power and reduce board temperature. By disabling the duty cycle, board is kept constantly ON in order to do lab testing and measurements. A valid value [0 - 1] must be provided by the user in config.h file for the DUTY_CYCLE_ENABLE field.

Parameters
[in]deviceA handle to the radar sensor device.
[in]flagAn unsigned 8-bit value, where '0' represents disable and non-zero value means enable duty cycle.
