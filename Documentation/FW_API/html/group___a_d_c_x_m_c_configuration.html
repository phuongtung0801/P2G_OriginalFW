Detailed Description

Functions and types to read and modify configuration of XMC's built in Analog/Digital Converter (ADC).

Adc_Configuration_t *  configuration 
)

This function returns the current ADC configuration.

This function returns the current ADC parameters of the XMC's built in Analog/Digital Converter through a struct of type Adc_Configuration_t. The struct must be provided by the user and may be uninitialized.

While automatic trigger is active, reading the current frame format may fail.

Parameters
[in]deviceA handle to the radar sensor device.
[out]configurationA pointer to a location, where the current ADC configuration will be stored.
Returns
An error code indicating if the function succeeded. The following error codes can occur:
  • RADAR_ERR_OK if the ADC parameters could be applied
  • RADAR_ERR_BUSY if the automatic trigger is active and the device is busy with acquiring radar data

uint16_t radar_set_adc_configuration ( Radar_Handle_t  device,
const Adc_Configuration_t *  configuration 
)

This function modifies the current ADC configuration.

The function changes the ADC parameters of the XMC's built in Analog/Digital Converter according to the provided structure of type Adc_Configuration_t.

A value check is applied to each field of pConfiguration. If any value or the combination of the values is out of range, the ADC parameters are not changed and an error is returned.

If the field uSamplingRate in pConfiguration is 0 the sampling rate will be set to the highest possible value.

The configuration can not be changed while the automatic trigger is active.

Parameters
[in]deviceA handle to the radar sensor device.
[in]configurationA pointer to the new ADC parameters to be applied.
Returns
An error code indicating if the function succeeded. The following error codes can occur:
  • RADAR_ERR_OK if the ADC parameters could be applied
  • RADAR_ERR_BUSY if the automatic trigger is active and the device is busy with acquiring radar data
  • RADAR_ERR_SAMPLERATE_OUT_OF_RANGE if the specified sampling rate is not supported by the ADC
  • RADAR_ERR_UNSUPPORTED_RESOLUTION if the specified ADC resolution is not supported by the ADC
