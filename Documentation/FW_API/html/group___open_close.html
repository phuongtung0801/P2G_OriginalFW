typedef void* Radar_Handle_t

This type identifies an open device.

A handle of this type is provided by the function radar_open_device. This handle must be provided to all other functions of this API to access that device.

◆ radar_open_device()

uint16_t radar_open_device ( int32_t  device_number,
Radar_Handle_t *  device 
)

This function opens an available radar device.

Before an device can be used, it must be opened with this function. If the function succeeds a handle of type Radar_Handle_t is returned that must be used in future function calls. When the device is not used any more, it must be closed by calling the function radar_close_device.

A driver may support multiple sensor devices at a time. The number of available devices can be retrieved by calling radar_get_number_of_available_devices.

Parameters
[in]device_numberThe number of the device to open. The first interface is 0.
[out]*deviceA pointer to a handle variable. If the device could be opened, the handle is stored to that variable.
Note
Due to performance issues, the API does not check provided device handles. Each function assumes that the provided device handle has been opened properly and that handles are not used any more after closing. If these assumptions are false this will lead to undefined behaviour.
Returns
An error code indicating if the device could be opened. The following error codes can occur:
  • RADAR_ERR_OK if the device could be initialized successfully
  • RADAR_ERR_DEVICE_DOES_NOT_EXIST if the device does not exist
  • RADAR_ERR_BUSY if the device has already been opened
