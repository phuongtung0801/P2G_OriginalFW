This function erase calibration data from the SRAM or Flash based on the target memory defined by argument Calibration_Target_t.

After clearing it resets the target memory with the default calibration values.

[in]targetTarget of the calibration defined by Calibration_Target_t.

◆ radar_save_calibration()

uint16_t radar_save_calibration ( Radar_Handle_t  device,
Calibration_Target_t  target,
Calibration_Data_t  calib_data_type,
const void *  calibration_data_ptr 
)

This function saves the calibration data to the SRAM or Flash based on the target memory defined by argument Calibration_Target_t.

All boards are pre-calibrated with TX power level, RX LNA gain, chirp-time, bandwidth etc... defined in the config.h file. Only for the change of TX power level, LNA gain disable or different Bandwidth and chirp-time requires new calibration.

Parameters
[in]deviceA handle to the radar sensor device.
[in]targetTarget of the calibration defined by Calibration_Target_t.
[in]calib_data_typeType of calibration data e.g. ADC data or Algorithm offset data defined by Calibration_Data_t.
[in]*calibration_data_ptrNULL pointer for ADC data while for Algorithm calibration data need to cast to Algo_Calibrations_t.
