P2G_FMCW

===============================================================================
Description of project
===============================================================================
Basic FMCW and Doppler Firmware with FMCW algorithm for Demo Position2Go board.

===============================================================================
Features
===============================================================================	
- Raw data acquisition for FMCW and Doppler supported
- Standard Range Doppler algorithm with tracking module
- Control PLL and BGT
- PGA gain control included
- MATLAB & C language communication interface for data extraction
- Radar GUI support
