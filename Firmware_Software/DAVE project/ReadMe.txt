===============================================================================
Folder Contents
===============================================================================

- P2G_FW
  *  FW ony DAVE™4 project that provides raw Radar ADC data based on Doppler or 
     FMCW modulation (configurable).
  *  Use this project to develop your own algorithm.

- P2G_FMCW
  *  DAVE™4 Project including FW described above (P2G_FW) and in addition adds 
     implementation of FMCW algorithm to measure and track distance, speed and 
     angle of multiple objects outdoor.
  *  Use this project in case you are interested in people tracking applications.
