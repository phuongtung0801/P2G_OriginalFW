◆ Callback_PGA_Level_t

typedef void(* Callback_PGA_Level_t) (void *context, int32_t protocol_handle, uint8_t endpoint, uint16_t pga_level_val)

This is the callback type for reading the Programmable Gain Amplifier (PGA) level.

Whenever a sensor board sends a message containing a PGA gain level value, a callback of this type is issued. The user must register the callback function by calling ep_radar_p2g_set_callback_pga_level.

Whenever a callback is issued, the following parameters are passed to the callback:

Parameters
[in]contextThe context data pointer, provided along with the callback itself through ep_radar_p2g_set_callback_pga_level.
[in]protocol_handleThe handle of the connection, the sending device is connected to.
[in]endpointThe number of the endpoint that has sent the message.
[in]pga_level_valThe gain level [0 - 7] of PGA used currently.

int32_t ep_radar_p2g_get_pga_level ( int32_t  protocol_handle,
uint8_t  endpoint 
)

This function reads out the programmable gain amplifier gain value, currently used in Position2Go demo kit.

There are eight possible gain levels, within [0-7] range. Each value corresponds to a binary gain value, please refer to the PGA112 data-sheet.

Parameters
[in]protocol_handleA handle to the connection to the sensor device.
[in]endpointThe endpoint in the connected device to be checked for compatibility.
[out]pga_levelThe PGA level currently used within the range [0 - 7].
Returns
If the specified endpoint is compatible to this implementation the function returns 0. If the endpoint is not compatible, a negative error code is returned.

◆ ep_radar_p2g_set_callback_pga_level()

void ep_radar_p2g_set_callback_pga_level ( Callback_PGA_Level_t  callback,
void *  context 
)

This functions registers a callback for radar Position2Go data messages.

If a callback for this message type is registered, that callback is issued every time, a connected board sends a message containing radar Position2Go data. If no callback is registered the message is ignored.

Connection and handle and endpoint number of the sending endpoint are passed to the callback along with the message data. Furthermore the pointer context set by this function is forwarded to the callback.

Parameters
[in]callbackThe function to be called when a radar frame data message is received.
[in]contextA data pointer that is forwarded to the callback function.

There are eight possible gain levels, one of them can be set by passing gain level argument [0-7] range.

If the specified endpoint is compatible to this implementation the function returns 0. If the endpoint is not compatible, a negative error code is returned.
