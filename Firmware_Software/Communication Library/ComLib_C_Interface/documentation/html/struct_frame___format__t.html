Detailed Description

This struct holds the parameters that define the structure of a radar data frame.

A struct of this type must be passed to ep_radar_base_set_frame_format to change the frame format. When the sensor board sends the current frame format, a structure of this type is returned through the callback Callback_Frame_Format_t. Use ep_radar_base_get_frame_format to query the current RF settings.
