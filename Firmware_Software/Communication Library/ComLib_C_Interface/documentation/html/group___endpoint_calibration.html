typedef void(* Callback_ADC_Calibration_Data_t) (void *context, int32_t protocol_handle, uint8_t endpoint, const float *calibration_data_ptr, uint16_t num_of_samples)

This functions registers a callback for raw ADC calibration data.

If a callback for this message type is registered, that callback is issued every time, a connected board requests raw ADC calibration data from device using ep_calibration_get_adc_flash_calibration_data() or .If no callback is registered the message is ignored.

Whenever a callback is issued, the following parameters are passed to the callback:

Parameters
[in]contextThe context data pointer, provided along with the callback itself through ep_calibration_set_callback_adc_calibration_data.
[in]protocol_handleThe handle of the connection, the sending device is connected to.
[in]endpointThe number of the endpoint that has sent the message.
[in]calibration_data_ptr*The radar ADC calibration data from FLASH or SRAM is stored in this memory location.
[in]num_of_samplesNumber of 16-bit samples received from device as calibration data.

typedef void(* Callback_Algo_Calibration_Data_t) (void *context, int32_t protocol_handle, uint8_t endpoint, const Algo_Calibrations_t *algo_calibration_data_ptr)

This functions registers a callback for algorithm offsets correction related calibration data.

If a callback for this message type is registered, that callback is issued every time, a connected board requests algorithm calibration data from device using ep_calibration_get_algo_flash_calibration_data() or ep_calibration_get_algo_sram_calibration_data().

If no callback is registered the message is ignored.

Whenever a callback is issued, the following parameters are passed to the callback:

Parameters
[in]contextThe context data pointer, provided along with the callback itself through ep_calibration_set_callback_algo_calibration_data.
[in]protocol_handleThe handle of the connection, the sending device is connected to.
[in]endpointThe number of the endpoint that has sent the message.

◆ ep_calibration_set_callback_adc_calibration_data()

void ep_calibration_set_callback_adc_calibration_data ( Callback_ADC_Calibration_Data_t  callback,
void *  context 
)

This functions registers a callback for radar ADC calibration data messages.

If a callback for this message type is registered, that callback is issued every time, a connected board sends a message containing radar calibration data. If no callback is registered the message is ignored.

Connection and handle and endpoint number of the sending endpoint are passed to the callback along with the message data. Furthermore the pointer context set by this function is forwarded to the callback.

Parameters
[in]callbackThe function to be called when a radar calibration data message is received.
[in]contextA data pointer that is forwarded to the callback function.
