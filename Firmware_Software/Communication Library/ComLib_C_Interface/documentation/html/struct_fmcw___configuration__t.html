Detailed Description

This structure holds the parameters to configure the RF behaviour in FMCW radar mode.

A structure of this type must be passed to ep_radar_fmcw_set_fmcw_configuration to change the FMCW settings. When the sensor board sends the current FMCW settings, a structure of this type is returned through the callback Callback_Fmcw_Configuration_t. Use ep_radar_fmcw_get_fmcw_configuration to query the current FMCW settings.
