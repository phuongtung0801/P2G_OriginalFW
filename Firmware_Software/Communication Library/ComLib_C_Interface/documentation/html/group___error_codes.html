Detailed Description

Protocol status codes.

If an error occurs during usage of the protocol or endpoint functions, one of the error codes defined here is returned. The error codes can be grouped in 4 classes.

1) General errors 2) Errors that occur when a connection can't be established. 3) Errors that occur when unexpected (or no) data is received from the connected device 4) Errors that occur due to improper endpoint request.
